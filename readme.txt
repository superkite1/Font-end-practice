一、移动web美团实战
1、项目的的lib下有移动端简单适配和淘宝移动端复杂适配两种适配方案,还有两种css初始化文件可参考,项目用的jquery没有用min.js
2、本地模拟的json数据再json文件夹下
3、项目采用模块化设计，有公共的部分再Common文件夹下、
4、查看效果建议开启服务端

二、vue仿电商项目
1、使用vue-router作为路由控制页面跳转
2、使用vue-awesome-swiper进行幻灯片以及滚动条的开发
3、使用scss取代原生css
4、使用webpack进行构建打包
5、使用fastclick、函数节流、localstorage存储数据、图片懒加载进行性能的优化
6、项目非常贴近真实开发环境，无论是scss文件的模块化分门别类以及第三方库的使用都非常具有借鉴意义

三、ES6彩票解析项目实战
1、使用npm+webpack+babel+gulp构建支持ES6的环境
2、使用express脚手架+mockjs做服务端接口数据
3、这个项目没有用框架、就是用了jquery\在class文件夹下面有ES6新特性的代码学习。回头可以看看、但也不能忘了MDN\MDN上面的还是比较全面的
4、项目厉害的地方在于gulp任务自动化前期代码的编写、就像模块化、前期准备很多工作、但开发的时候会变得非常方便、高效、这些任务自动化、比如说文件监听实时修改并且自动触发浏览器刷新这个实在是太方便了:D
5、在业务方面其实还有一个厉害的地方彩票项目的核心，就是是重现排列组合的算法、这个是高中学的，但从来没有用代码实现过、需要多看几遍。
